== Description ==
A WordPress plugin for caste/family-based registration and display with frontend and backend functionality.

== Installation ==
1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the shortcodes `[cd_registration]` and `[cd_family_listing]` to display the form and listing.
